# Electric-Car-Market-Insights-Excel-Dashboard
Interactive Excel dashboard analyzing electric car data to identify trends, key metrics, and insights using charts and pivot tables.
 Electric Car Market Insights | Excel Dashboard

 Project Overview
 This project focuses on analyzing electric car data using Microsoft Excel and presenting insights through an interactive dashboard.  
 The dashboard helps understand trends, comparisons, and key metrics related to electric vehicles.

 Dashboard Features
- Interactive charts and visuals  
- Cleaned and structured dataset  
- Easy-to-understand KPIs  
- Filters for better data exploration  

 Tools Used
- Microsoft Excel  
- Excel Charts & Pivot Tables  
- Basic Data Cleaning Techniques  

 Dataset
 The dataset contains information related to electric cars such as:
- Vehicle details  
- Sales or performance metrics  
- Category-based comparisons  

 Objective
- Analyze electric vehicle data efficiently  
- Convert raw data into meaningful insights  
- Build a visually appealing and informative dashboard

How to Use
1. Download the Excel file from the repository  
2. Open it using Microsoft Excel  
3. Use filters and visuals to explore the data
